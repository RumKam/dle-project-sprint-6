# Проект: Pretrain и SFT для русскоязычной LLM

## Описание задачи

В рамках проекта выполнена реализация двух ключевых этапов создания языковой модели: **предобучение (pretrain)** на корпусе русской литературы и **дообучение с учителем (SFT)** на инструктивном датасете.  
Цель — отработать на практике полный цикл обучения LLM: от сбора и предобработки данных до тонкой настройки модели и оценки качества генерации.

## Цели проекта

- **Pretrain**  
  Обучить модель с архитектурой decoder-only (≈150M параметров) продолжать фразы в стиле русской литературы.  
  Использован корпус произведений из репозитория `RussianNovels`.  

- **Post-train SFT**  
  Адаптирована модель `Qwen2.5-0.5B` к диалоговому формату и инструкциям на русском языке.  
  Использован датасет `d0rj/alpaca-cleaned-ru`.  

## План проекта

### Этап 1. Pretrain

1. Скачаны все тексты из репозитория `RussianNovels/corpus` и объединены в единый датасет
2. Проведён препроцессинг: удалены дубликаты, предложения с не-кириллицей, нормализована пунктуация, удалены управляющие символы и исторические буквы
3. Текст токенизирован и разбит на чанки длиной 512 токенов с добавлением `<bos>`/`<eos>`
4. Обучен BPE-токенизатор на 3000 токенов
5. Данные токенизированы и представлены в виде `datasets.Dataset`
6. Инициализирована модель decoder-only с парметрами hidden_size=1024, intermediate=1536, layers=16, heads=16, kv_heads=8 — 132 млн параметров
7. Обучение запущено с помощью `SFTTrainer` с batch_size=16, learning_rate=1e-4, weight_decay=0.01, max_length=512, 3 эпохи, fp16
8. Добавлены коллбэки для валидации на `test_prompts`
9. Сгенерированы ответы на тестовые промпты

### Этап 2. Post-train SFT

1. Загружен датасет `d0rj/alpaca-cleaned-ru`
2. Датасет преобразован в формат диалогов: `user` → `assistant`
3. Загружена базовая модель `Qwen2.5-0.5B`
4. Настроен `SFTTrainer` с batch_size=8, learning_rate=2e-5, max_length=256, 3 эпохи, fp16
5. Модель дообучена на инструктивном датасете 
6. Выполнена генерация ответов на контрольные вопросы, модель отвечает связно на русском языке
